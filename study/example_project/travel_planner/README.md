# TravelMate 미니 프로젝트

JS 기초 학습용 프로젝트입니다. 사용자는 여행 정보를 입력하고 디데이, 준비물, 예산 등을 확인할 수 있습니다.

## 사용 기술
- HTML5 / CSS3
- JavaScript ES6 (let/const, 객체, 배열, 함수, 이벤트)

## 기능 요약
- 여행 D-Day 계산
- 준비물 체크리스트
- 여행 예산 계산기
